# 개발 1조 – Personal Introduction App

## 프로젝트 개요
저희 개발 1조는 4명으로 구성되어 **개인 소개 페이지를 제공하는 Flutter 앱**을 제작했습니다.  
각 팀원이 개인 페이지를 만들고, Git 관리와 발표 자료(PPT) 작업을 역할 분담하여 개발을 진행했습니다.

---

## 주요 기능
- 팀원별 개인 페이지 제공  
- Git 충돌 문제 해결 및 브랜치 관리 적용  
- 반응형 UI 적용으로 작은 화면에서도 정상 동작  

---

## 기술 스택
- **프레임워크:** Flutter  
- **언어:** Dart  
- **버전 관리:** Git / GitHub  

---

# 개발 중 문제 및 해결
- **Git 충돌 문제**: 브랜치 전략 정리, Compare Code 활용, pull/push 루틴 적용
- **Widget 구조 혼동**: Column/Row, Container/SizedBox, child/children 개념 학습 후 적용
- **UI overflow 문제**: 반응형 구조, margin/padding, SingleChildScrollView, ListView, Expanded 적용
